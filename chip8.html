Summary

Interpreter of Chip8(http://en.wikipedia.org/wiki/Chip8)
What distincts this interpreter from others is that it uses opcodes from programs to control the frequency of notes generated by synthesizer.
Opcodes in Chip8 are formed from four numbers each in range 0-15. Each of these four numbers determines the frequency generated by one of four oscillators.
User can change waveshapes of these oscillators and their volume.

Instructions

Extract the archive and run chip8.jar.
From the menu "actions" choose "load program" and choose file from Chip-8 demos. The program will start with default settings for oscillators
To change the sounds of oscillators before running the program choose the "settins" options from "actions" menu
Option "toggle synesthesia" turns off the sound generation. Programs will run faster with sound generation turned off (when sound is synthesized the programs are deliberately slowered because otherwise the generated sounds would be too short).
To load other program turn off application.
